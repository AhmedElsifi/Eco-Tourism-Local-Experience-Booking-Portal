# 🌍 Eco-Tourism & Local Experience Booking Portal

A web-based platform that connects travelers with local guides to book authentic eco-tourism experiences.

The system focuses on promoting sustainable tourism by enabling users to explore local cultures, book guided tours, and share feedback through reviews and ratings.

---

## 🚀 Features

* User registration and login
* Tour and experience browsing
* Booking system
* Multilingual guide profiles
* Reviews and ratings

---

## 🧰 Tech Stack

* PHP (OOP)
* MySQL
* MVC Architecture
* XAMPP / Apache

---

## ⚙️ Setup

1. Clone the repository
2. Create a database named `eco_tourism`
3. Configure the database connection in `/core/connection.php`
4. Run the project using XAMPP

---

## 📌 Notes

* Passwords are securely hashed using `PASSWORD_DEFAULT`
* PDO is used for database interactions
* Follow the project guidelines
